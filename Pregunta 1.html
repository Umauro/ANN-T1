El problema a enfrentar es el de clasificacin entre dos clases mutuamente excluyentes. Por lo tanto, a partir de un vector se debe obtener un valor categrico, lo que de utilizar Redes Neuronales Feed Forward se puede lograr mediante la aplicacin de transformaciones entre capas de neuronas, donde finalmente la capa de salida cuente solo con dos neuronas, cuyo resultado de computo entregar la probabilidad de pertenecer a cierta etiqueta o no. Al ser etiquetaas mutuamente excluyentes se debe utilizar una funcin de activacin adecuada, como por ejemplo la funcin Soft-Max, lo que entregar ambos resultados entre 0 o 1, donde adems la suma entre ellos da 1. Finalmente, para entregar un valor entre 0 y 1, se determina cual de las probabilidades entregadas es mayor, y en funcin de eso se retorna la etiqueta de la clase requerida.

Cabe destacar que para resolver este tipo de problemas se busca aproximar $P(y|x)$, es decir, la pertenencia a una determinada clase dado un ejemplo en especfico. Por lo anterior, se requiere utilizar una funcin de perdida adecuada en el entrenamiento, como por ejemplo la Cross-Entropy Loss, la cual calcula las divergencias entre la funcin de probabilidad que se desea aproximar y la aproximacin. Ac surge un nuevo problema en este caso, pues al tener clases desbalanceadas el clasificador tender a clasificar un mayor cantidad de ejemplos con la etiqueta de la clase dominante, debido a que la Cross-Entropy Loss funciona en base a accuracy, y al clasificar todo con la etiqueta de la clase dominante este valor tendera a ser ms alto si es desbalance es demasiado. Finalmente compare a travs de una mtrica de desempeo sobre el conjunto de pruebas, en este caso como trabajamos un problema desbalanceado, mida f1 score weighted, comente sobre esta decisin Es esperable la diferencia entre relu y sigmoidal en base a los grficos realizados?

Para evaluar ambas redes se utiliza el f1 score, el cual trata de considerar la accuracy y el recall. La precisin, en un problema de clasificacin binario, se puede ver como la razn entre los verdaderos positivos y todos los valores entregados como positivos por el modelo. El recall en cambio, se puede ver como la razn entre los verdaderos positivos entre todos los valores verdaderos del conjunto. En problemas desbalanceados, si se considera solo la accuracy para evaluar los distintos modelos, se tender a elegir un modelo que clasifique la gran mayora de los datos con los valores de la clase dominante, en cambio si se utiliza solo el recall para elegir el modelo, dicho modelo tender a clasificar todos los catos como la clase menos representada. Es por lo anterior que se utiliza el F1-Score, pues considera la media armnica entre ambas mtricas.

$$ F1_{score} = 2\cdot\frac{precision \cdot recall}{precision + recall}$$

Considerando los grficos anteriores de ambas redes, es de esperar que la red con activacin sigmoidal obtenga mejores resultados. Pues en el caso de la red con activacin Relu, si se observa que el valor de la Loss es muy alto es porque $P(y|x)$ y $f(x)$ divergen mucho entre si.

def decay_experiment(iterations=3,lr=0.01):
    n_decay = 10
    lear_decay = np.logspace(-6,0,n_decay)
    decay_sigmoid_dict = dict()
    decay_relu_dict = dict()
    model_decay_sigmoid_dict = dict()
    model_decay_relu_dict = dict()
    for decay in lear_decay:
        sigmoid_list = []
        relu_list = []
        for i in range(3):
        # Training FF with sigmoid activation
            model_sigmoid = Sequential()
            model_sigmoid.add(Dense(256, input_dim=X_train_scaled.shape[1],kernel_initializer='uniform',activation='sigmoid'))
            model_sigmoid.add(Dense(1, kernel_initializer='uniform',activation='sigmoid'))
            model_sigmoid.compile(optimizer=SGD(lr=lr,decay=decay),loss='binary_crossentropy')
            hist_sigmoid = model_sigmoid.fit(X_train_scaled,y_train,epochs=100,verbose=0,validation_data=(X_val_scaled,y_val))
            sigmoid_list.append(hist_sigmoid)
            # Training FF with relu activation
            model_relu = Sequential()
            model_relu.add(Dense(256, input_dim=X_train_scaled.shape[1],kernel_initializer='uniform',activation='sigmoid'))
            model_relu.add(Dense(1, kernel_initializer='uniform',activation='sigmoid'))
            model_relu.compile(optimizer=SGD(lr=lr,decay=decay),loss='binary_crossentropy')
            hist_relu = model_relu.fit(X_train_scaled,y_train,epochs=100,verbose=0,validation_data=(X_val_scaled,y_val))
            relu_list.append(hist_relu)

        decay_sigmoid_dict[str(decay)] = sigmoid_list
        decay_relu_dict[str(decay)] = relu_list
        model_decay_sigmoid_dict[str(decay)] = model_sigmoid
        model_decay_relu_dict[str(decay)] = model_relu
    return decay_sigmoid_dict, decay_relu_dict, model_decay_sigmoid_dict, model_decay_relu_dict
